1- Crie uma página com uma pergunta: “Que fruta é essa?”. A página deve conter também uma figura da sombra de uma fruta. Um campo de texto (formulário) para o usuário digitar a resposta e um botão para verificar a resposta. Ao digitar um nome no formulário e clicar no botão, verifique se o usuário digitou o nome da fruta. Se acertar de os parabéns. Se não informe a resposta correta.


Que fruta é essa?


2- Crie uma página com uma pergunta: “Que fruta você mais gosta?”.A página deve ter uma imagem de uma (?) interrogação, um formulário e um botão. No formulário o usuário deverá entrar com um nome de uma fruta (diversas). Ao clicar no botão, altere o src da imagem para exibir a imagem correspondente à fruta escolhida pelo usuário. Caso não exista a fruta, mantenha a interrogação e informe uma mensagem ” vamos cadastrar sua fruta favorita.”


3- Crie uma página que mude dinamicamente seu estilo de acordo com o horário do sistema.

Se o usuário acessar a página pela manhã, deve ser mostrado “Bom Dia”, deve apresantar à hora e minutos atuas, a cor do “background” deve ser uma cor mais clara para representar ‘dia' e uma imagem relacionada a uma manhã de sol também deve ser apresentada. Essas caracteriscas devem também acontacer para acesso a ‘tarde’ e ‘noite'. Quem quiser, pode fazer para madrugada. Deve ser usado o IF ELSE para essas condicinais. Para capturar a data atual pode ser usado a função new Date() e getHours() e getMinutes() https://www.w3schools.com/jsref/jsref_gethours.asp. Para a estrutura use o HTML e o CSS para ficar bonitinho.
